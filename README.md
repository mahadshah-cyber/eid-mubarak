# 🌙 Eid Mubarak Website

A beautiful and animated **Eid Mubarak greeting website** built using **HTML, CSS, and JavaScript**.  
This project features a modern Islamic-themed design with stars, moon, lanterns, and a customizable message panel.

---

## ✨ Features

- 🌌 Animated starry background
- 🌙 Floating moon effect
- 🏮 Swinging lanterns (left & right)
- ✨ Falling particle animations
- 🖋 Elegant typography (Google Fonts)
- 🧾 Custom message panel
- 💾 Download your customized greeting as HTML

---

## 🎨 Customization Panel

Click the **✦ button** (bottom-right corner) to:
- Change your name
- Edit Arabic text
- Modify the greeting message
- Download your personalized Eid card
